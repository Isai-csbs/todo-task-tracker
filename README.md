# Full-Stack To-Do Task Tracker

An enterprise-grade, lightweight task manager designed to demonstrate complete CRUD (Create, Read, Update, Delete) processing using a native PHP API backend and dynamic asynchronous JavaScript.

## Features
* **RESTful JSON API:** Configured endpoints running HTTP methods securely.
* **Robust Data Sanitization:** Shielded against database injections and cross-site scripts.
* **Client-Side Virtualization:** Graceful runtime fallback using mock array updates when hosted on static services like GitHub Pages.
